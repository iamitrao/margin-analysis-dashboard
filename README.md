🍬 Nassau Candy Profitability Dashboard
📌 Overview
End-to-end analysis of product profitability and margin performance for Nassau Candy Distributor. The project identifies high-profit products, margin risks, and division-level inefficiencies using data-driven insights.
📊 Features
Product-level profit & margin analysis
Division performance comparison
Cost vs Sales diagnostics
Pareto (80/20) profit analysis
Interactive Streamlit dashboard
🛠️ Tech Stack
Python, Pandas, Matplotlib, Streamlit
🚀 Run the App
pip install -r requirements.txt
streamlit run app.py
